## 📖 Overview

A focused implementation of **Support Vector Machines (SVM)** for both classification and regression tasks. This project explores SVM theory, kernel tricks, hyperparameter tuning, and practical applications on real-world datasets.

---

## 🎯 What's Covered

- SVM theory — maximum margin hyperplane, support vectors
- Linear SVM for binary and multi-class classification
- Kernel functions: Linear, RBF (Gaussian), Polynomial, Sigmoid
- SVR (Support Vector Regression)
- Hyperparameter tuning: C, gamma, kernel via GridSearchCV
- Visualization of decision boundaries

---

## 📊 Datasets Used

| Dataset | Task | Classes |
|---------|------|---------|
| Iris | Multi-class classification | 3 |
| Breast Cancer | Binary classification | 2 |
| Wine | Multi-class classification | 3 |
| Custom regression | SVR | Continuous |

## 📈 Results

| Kernel | Iris Accuracy | Breast Cancer Accuracy |
|--------|-------------|----------------------|
| Linear | 97.3% | 96.5% |
| RBF | 98.0% | 97.4% |
| Polynomial | 96.7% | 95.8% |
